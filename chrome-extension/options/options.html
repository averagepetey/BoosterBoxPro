BoosterBoxPro – API URL

Set the backend API URL the extension uses to load box data. Default is your local server.

No trailing slash. Examples: http://localhost:8000 or https://your-app.run.app
Saved.

If you see “Connection error”

  1. Start the BoosterBoxPro backend (e.g. python main.py in the project folder).
  2. Confirm it’s running at the URL above (e.g. open http://localhost:8000/docs in your browser).
  3. Reload the TCGplayer tab and click Retry in the extension panel.
